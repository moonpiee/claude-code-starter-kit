# Project Overview

This repository is a production-ready starter kit for Claude Code projects. It helps teams move from idea to execution with structured planning files, reusable prompts, and example implementations.

# Objectives

- Make project setup repeatable and clear
- Reduce ambiguity in communication with Claude Code
- Give developers a practical system for documenting goals, architecture, tasks, and decisions
- Provide strong examples that can be copied into new projects

Success Criteria

# Coding Standards

- Prefer clear, practical documentation over filler
- Use concise language and concrete examples
- Keep templates actionable and easy to fill in
- Do not hardcode secrets or credentials
- Preserve a useful structure even when examples are simplified

# Folder Structure

- `/prompts` contains reusable prompting patterns
- `/examples` contains filled-in sample projects
- `/screenshots` is for visual references

# Commands

- Review repository status with `git status`
- Preview documents in the editor

# Architecture Notes

This project is documentation-first. The core value comes from well-structured content rather than runtime code. The examples demonstrate how the framework can support different product types.
